# Lost and Found Item Tracking System

This is my first GitHub project.
It is a desktop application developed using Python, PyQt5, and SQLite.

## 📌 Project Purpose

The aim of this project is to track lost and found items in a simple and organized way.

⚠️ PROJECT STATUS: This project is currently under development (Work in Progress).

## ⚙️ Technologies Used

* Python
* PyQt5
* SQLite

## 🚀 Features

* Add lost items
* Add found items
* Search items
* User-friendly interface

## ▶️ How to Run

1. Install Python
2. Install required libraries:
pip install PyQt5
3. Run the main file:
python main.py
